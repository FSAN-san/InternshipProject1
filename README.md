# About
This is a `React18 + TypeScript + Vite` project.

The plugin is as follows:
1. `Axios`
2. `ESLint`
3. `@reduxjs/toolkit`
4. `Redux`
5. `React Redux`
6. `PostCSS`
7. `SCSS`
8. `Ant Design`
9. `React Router Dom`
10. `@ant-design/icons`

# other
The antd introduction mode is automatically introduced on demand
